# HelsinkiPlaces

This is a Java + React.js application with using Spring.

At the backend the application download the information about places from MyHelsinki Open API. Then it will list it in a browser.

To run this application:
  1. run MyApplication.java
    -it will ask you for tags if they needed.
  2. in folder my-app open terminal and write "npm start"
  
 It will open a page on localhost:3000 and show the data in a table.
